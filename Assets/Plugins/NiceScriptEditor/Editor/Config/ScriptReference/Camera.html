OnPostRender: Event function that Unity calls after a Camera renders the scene. OnPreCull: Event function that Unity calls before a Camera culls the scene. OnPreRender: Event function that Unity calls before a Camera renders the scene. OnRenderImage: Event function that Unity calls after a Camera has finished rendering, that allows you to modify the Camera's final image. OnRenderObject:OnRenderObject is called after camera has rendered the Scene. OnWillRenderObject:OnWillRenderObject is called for each camera if the object is visible.
